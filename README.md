# Статистика филиалов МФЦ

Приложение для анализа статистики операций по филиалам МФЦ.

## Функции

### Вкладка "Восстановление"
- Анализ одного CSV файла
- Подсчет количества операций по филиалам
- Экспорт результатов в Excel

### Вкладка "Создание + Подтверждение"
- Анализ двух CSV файлов одновременно
- Суммирование статистики из двух файлов
- Экспорт суммарных результатов в Excel

## Установка

### Способ 1: Установка из исходного кода


# Клонировать репозиторий
git clone <repository-url>
cd mfc-stats-app

# Установить зависимости
pip install -r requirements.txt

# Установить приложение
pip install .

# Запустить приложение
mfc-stats-gui

## Системные требования
Linux (Ubuntu 18.04+, Fedora 30+, CentOS 8+)

Python 3.6 или выше

512 MB оперативной памяти

100 MB свободного места на диске

Формат файлов
Приложение поддерживает CSV файлы со следующими характеристиками:

Разделитель: точка с запятой (;)

Кодировка: автоматическое определение

Минимум 2 столбца (второй столбец содержит названия филиалов)
